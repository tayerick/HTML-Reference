This project is my demonstration of html tags and what they do.  Most of it is semantic while bits of it
are presentational (mainly the deprecated tags).  This project aims to help further my understanding of html
and how it works with the Internet.  It consist of eight files: index.html, text.html, media.html, forms.html,
tables.html, structure.html, deprecated.html, experiments.html.  Start at the index.html from where you can navigate
to the other pages.  Please have fun with it!

In my experiments page, I decided that my three experiments will demonstrate the following: Attribute Logic:

1) When you set the optimum attribute to a value higher than the high threshold, you are defining the "ideal" state of
   the data at the top of the scale. The browser interprets this mathematically: since the goal is to be near the optimum,
   it renders the bar green when the value is high, yellow when it dips into the middle range, and red when it falls below
   the low marker.  It's important to note this as the browser has its own settings for these attributes.

2) The second experiment test how many nested <section> tags the browser can support before it fails.  What you'll notice
   is that depending on the browser, the nested tags may or may not trigger the browser to freeze.  Modern browsers have protocols
   in place to prevent this.

3) This next expirment demonstrates some of the functionality that HTML allows, but the issue that is most noticeable with this
   is that, depending on the browser, the browser may or may not allow the key combos dictated by your html.  In this experiment, using
   chrome kept me from using ALT + F to select the fire option.
